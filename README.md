# Bluestaq Elevator Take Home Challenge

This program simulates an elevator, handling passenger 
boarding and exiting from different floors.

**Assumptions made:**
- All passengers only rides on elevator once.
- No new Passenger enters the simulation after it begins.
- Passengers always board if there is space in the elevator.
- The building has a fixed number of floors.

**Unimplemented features:**
- Multiple elevators.
- Optimized scheduling algorithms.

**How to run**
```bash
javac -d out src\Main.java src\model\*.java
java -cp out Main
```
